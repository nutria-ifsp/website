# 🍏 Nutria - Landing Page

> O ambiente digital perfeito para a sua alimentação. Encontre uma comunidade, compartilhe receitas e desenvolva sua rotina com saúde e motivação.

## 💻 Sobre o Projeto

A Landing Page do Nutria foi desenvolvida para apresentar o aplicativo e disponibilizar seu download. A página é responsiva, conta com alternância de tema (Claro/Escuro) e modais interativos para gerenciar o download dependendo do dispositivo do usuário (somente Android).

## ✨ Funcionalidades da Página

- **Alternância de Tema (Dark/Light Mode):** Salva a preferência do usuário automaticamente.
- **Detecção de Dispositivo:** Entrega o download direto do APK para Android ou exibe instruções via QR Code para Desktop.
- **Carrosséis Interativos:** Navegação fluida para visualização dos planos de assinatura e membros da equipe.
- **Design Responsivo:** Adaptado para celulares, tablets e telas grandes.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando:

- **HTML5** (Semântico e estruturado)
- **CSS3** (Arquitetura modularizada e variáveis de cor)
- **JavaScript (Vanilla)** (Lógica dividida em múltiplos arquivos para melhor manutenção)
